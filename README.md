## Usage

Install package:

```sh
# npm
npm install @op-ent/unstyled-ui-theme

# yarn
yarn install @op-ent/unstyled-ui-theme

# pnpm
pnpm install @op-ent/unstyled-ui-theme
```

Import:

```js
// ESM
import { sum } from '@op-ent/unstyled-ui-theme'

// CommonJS
const { sum } = require('@op-ent/unstyled-ui-theme')
```
